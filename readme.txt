BuddyForms Anonymous Author

== Description ==

Select a default Author for every form and give your users the option to publish Anonymous under the default Author.

All public data like username user link avatar will be displayed with the anonymous author profile.

The real Author can still access edita dn delete the post or turn off anonymous author and make the post below to him in public at any time.

== Installation ==

You can download and install the plugin by using the built-in WordPress plugin installer. If you download the plugin manually,
make sure it is uploaded to "/wp-content/plugins/".

Activate the plugin in the "Plugins" admin panel by using the "Activate" link.
